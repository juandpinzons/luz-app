# LEARNING_PARTNERSHIP_POLICY.md

**Status:** Draft v1.0\
**Applies to:** LUZ Alpha-0\
**Owner:** Product / Engineering

------------------------------------------------------------------------

# Purpose

The purpose of the Alpha-0 program is to validate and improve the
intelligence architecture of LUZ through collaboration with a limited
group of early users.

Participants are not training a general-purpose AI model. Instead, they
are helping improve the behavior of LUZ's internal intelligence systems,
including Memory, Knowledge, Context, Presence, and conversation
quality.

Participation is voluntary and based on informed consent.

------------------------------------------------------------------------

# Guiding Principles

1.  Human-first.
2.  Privacy by design.
3.  Transparency.
4.  User control.
5.  Data minimization.
6.  Continuous improvement.
7.  No advertising use.
8.  No sale of personal information.

------------------------------------------------------------------------

# What May Be Collected During Alpha

## Conversation Data

-   User messages.
-   LUZ responses.
-   Conversation timestamps.
-   Session metadata.

## Memory Evaluation

-   Memories captured.
-   Memories ignored.
-   Memory ranking.
-   Memory retrieval success.
-   User corrections.

## Knowledge Evaluation

-   Patterns detected.
-   Inferences generated.
-   Confidence scores.
-   User validation or correction.

## Presence Evaluation

-   Tone effectiveness.
-   Helpfulness.
-   Intrusiveness.
-   Timing.
-   Continuity quality.

## Explicit Feedback

Examples: - 👍 Helpful - 👎 Not helpful - "Incorrect" - "Already knew
this" - "Very useful"

------------------------------------------------------------------------

# What This Data Is Used For

Collected information may be used to:

-   Improve Memory Engine heuristics.
-   Improve Knowledge Engine reasoning.
-   Improve Context assembly.
-   Improve Presence behaviors.
-   Reduce false memories.
-   Improve prioritization and retrieval.
-   Improve overall user experience.

The information is **not** intended to train a general-purpose language
model.

------------------------------------------------------------------------

# User Consent

Participation requires explicit consent.

Users should be able to:

-   Join the Alpha program.
-   Leave the Alpha program.
-   Request deletion of their Alpha data.
-   Disable learning participation at any time.

------------------------------------------------------------------------

# Transparency

Whenever practical, LUZ should explain when a conclusion is based on
previous conversations rather than the current message alone.

------------------------------------------------------------------------

# Error Learning

Corrections are valuable signals.

Example lifecycle:

Prediction ↓ User correction ↓ Knowledge refinement ↓ Future improvement

------------------------------------------------------------------------

# Privacy Commitments

LUZ commits that Alpha data will not be:

-   Sold.
-   Used for advertising.
-   Shared with third parties for marketing.
-   Used outside the stated purpose of improving LUZ.

------------------------------------------------------------------------

# Temporary Feature

Enhanced telemetry is an Alpha-only capability.

Expected lifecycle:

Alpha-0 ↓ Architecture validation ↓ Engine refinement ↓ Reduced
telemetry ↓ Production behavior

As LUZ matures, the system should progressively rely on its architecture
rather than extensive evaluation telemetry.

------------------------------------------------------------------------

# Success Criteria

The Alpha program is successful if it enables measurable improvements
in:

-   Memory relevance.
-   Knowledge accuracy.
-   Context quality.
-   Presence.
-   User trust.
-   Long-term continuity.

------------------------------------------------------------------------

# Philosophy

Early users are not merely testers.

They are collaborators helping shape a new form of personal
intelligence.

Their participation helps LUZ become a more thoughtful, respectful, and
genuinely useful long-term companion while preserving transparency,
consent, and user control.
