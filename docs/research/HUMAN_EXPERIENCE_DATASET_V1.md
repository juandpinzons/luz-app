# HUMAN EXPERIENCE DATASET V1

## Purpose

The purpose of this dataset is **not** to train an AI model.

Its purpose is to evaluate whether **LUZ** truly understands human
beings over time.

The dataset is intended to validate:

-   Memory
-   Knowledge
-   RealitySnapshot
-   Context
-   Presence
-   Conversation

Each profile should evolve naturally over months and years.

# Success Criteria

LUZ should be able to:

-   Remember important events months later.
-   Detect personal growth.
-   Detect recurring struggles.
-   Adapt its tone over time.
-   Preserve long-term context.
-   Connect related memories.
-   Respect user autonomy.
-   Demonstrate genuine care without becoming intrusive.

------------------------------------------------------------------------

## Vision

This is not a testing dataset.

It is a library of human lives designed to evaluate whether LUZ truly
understands people before the public beta.
